Account Management

accounts

Retrieves an authentication token to perform auth'd requests.

POST /accounts/login
LoginRequest

Contains the username and password of the account being authenticated.

Request Content-Types: application/json
Request Example
{
  "username": "johna",
  "password": "helloworld1!"
}
200 OK
LoginResponse

The credentials given were valid.

400 Bad Request
ErrorResponse

A required parameter is missing.

401 Unauthorized
ErrorResponse

The credentials given were not valid.

500 Internal Server Error
ErrorResponse

Something went wrong on the server.

Icon Requests

requests

Gets all icon requests for an app (icon pack).

GET /requests

The authenticated user must be the owner or a collaborator.

key
in query
string

The key of the app (icon pack) to get requests for.

offset
in query
integer

The offset, used for pagination. Defaults to 0.

limit
in query
integer

The limit, used for pagination. Defaults to 24.

q
in query
string

The query string, used to search for specific requests.

states
in query
string

The comma-separated list of states to filter by. 0 = new, 1 = completed, 2 = implemented, 3 = hidden.

200 OK
RequestsResponse

Icon requests were retrieved.

400 Bad Request
ErrorResponse

Invalid parameters were received.

404 Not Found
ErrorResponse
